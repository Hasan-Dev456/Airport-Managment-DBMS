# Airport Management Database System

## Overview

The Airport Management Database System is a relational database project developed as part of a Database Management Systems course. The project models the core operations of an airport, including aircraft management, flights, employees, maintenance, parking, hangars, technicians, and air traffic controllers.

The database is designed using normalization principles and SQL constraints to ensure data integrity and efficient querying.

---

## Features

- Relational database design
- Entity Relationship Diagram (ERD)
- Normalized database schema
- Primary and Foreign Key constraints
- Sample data for testing
- SQL queries for retrieving useful information
- Airport employee and technician management
- Aircraft maintenance and testing records
- Flight management
- Hangar and parking management
